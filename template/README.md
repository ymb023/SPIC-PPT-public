## 设计原则：固定头部 + 灵活主体

**这是v2与v1的核心区别**。每一页内页都遵循统一的"头部三件套"：

```
┌──────────────────────────────────[LOGO]──┐
│                                            │
│  深蓝大标题                                 │  ← 1. 标题（深蓝加粗）
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━           │  ← 2. 集团固定款细波浪线
│                                            │
│  ┌─虚线绿框────────────────────────────┐  │  ← 3. 关键论点（带红色数字）
│  │ 总体性/描述性句子，红色关键数字突出   │  │
│  └────────────────────────────────────┘  │
│                                            │
│  [主体内容 · 灵活适配各种版式]              │  ← 4. 主体（六种典型版式）
│                                            │
│  数据来源：xxx                  页码         │
└────────────────────────────────────────────┘
```

主体可以是：图表、表格、卡片、列表、时间线、数据墙——按内容选合适的版式。

## 快速开始

### 1. 复制空白骨架

```bash
# 在你的项目目录下
cp template.html ../我的新项目/index.html
# CSS、_assets/、_shared/ 保持原路径引用即可，不要复制
```

### 2. 改三件套

打开 `index.html`，按顺序改：

1. **改封面**：`<h1 class="cover-title">` 主标题 + `cover-author` 汇报人和日期
2. **改章节分隔页**（如有）：`section-num` 编号 + `section-title` 章节名
3. **改内页**（复制内页 `<section class="slide">` 块）：
   - `page-title` → 页面主标题（深蓝大字）
   - `key-point` → 关键论点（红色数字用 `<span class="accent">数字</span>`）
   - 主体内容 → 从 6 种版式中选合适的（见 template.html 注释）
   - `data-source` → 数据来源
   - `page-num` → 页码

### 3. 导出 PDF

```bash
chrome --headless=new --no-pdf-header-footer \
  --print-to-pdf=output.pdf \
  --virtual-time-budget=15000 \
  "file:///path/to/index.html"
```

或浏览器打开 → Ctrl+P → 保存为PDF。

## 设计原则

### 必须遵守的VI规范

1. **LOGO必须用官方文件**——来自 `../_shared/logos/etrc/etrc-logo.png`
2. **字体必须微软雅黑**（中文）+ Arial（西文）
3. **集团固定款波浪飘带不能省**——封面 `cover-wave` + 内页 `page-wave` 是核心识别元素
4. **二级单位用横排"国家电投经研院 SPIC ETRC"标识**——已是默认

### 经研院专家惯例

1. **关键论点用虚线绿框包裹**——使用 `<div class="key-point">`，不要用纯文字
2. **红色数字突出**——关键数据用 `<span class="accent">3.3%</span>`，引导阅读
3. **数据来源严谨标注**——含数据的页面底部必加 `<div class="data-source">`
4. **图表必须有绿色边框**——使用 `<div class="chart-frame">` 包裹
5. **表格双色标题**——前面绿底 `<th>`，"说明"列用 `<th class="col-orange">`

### 排版铁律

1. **左对齐为主**——标题、正文都左对齐；只有封面/章节大字居中
2. **每页1个核心论点 + 1种主体版式**——不要堆叠
3. **每页留白≥30%**——避免塞满
4. **字号梯度清晰**——封面36pt → 页面标题26pt → 论点13pt → 正文11pt → 图注9pt
