### 主题官方配色（theme1.xml 抽取）

| 角色 | 色值 | 用途 |
|---|---|---|
| 主背景 | `#FFFFFF` 白 | 内容页底色 |
| 主文字 | `#000000` 黑 | 正文 |
| 主强调色 | `#0082D2` 蓝 | accent1，电投蓝主调 |
| 二强调色 | `#2DB56B` 绿 | accent2，电投绿（对应LOGO双螺旋） |
| 三强调色 | `#2BADFF` 浅蓝 | accent3，辅助 |
| 四强调色 | `#84C945` 嫩绿 | accent4，辅助 |
| 中性灰蓝 | `#778495` | dk2，次级文字 |
| 浅灰底 | `#F0F0F0` | lt2，分隔/底色 |
| 中性灰 | `#8792A1` / `#ADB5BF` | 辅助灰 |

### 实际使用中的扩展色（高频）

| 色值 | 用途 | 典型场景 |
|---|---|---|
| `#C00000` 暗红 | **关键论点强调** | 正文里需要引导读者眼球的关键句 |
| `#002060` 深蓝 | 副标题/讲者署名 | 封面讲者署名用色 |
| `#00B050` 鲜绿 | 章节大字（CONTENTS等） | 目录页标识 |
| `#92D050` 嫩绿 | 编号/装饰文字 | "目录"两字竖排 |

**配色组合策略**：
- 主色：电投蓝 `#0082D2` 或 电投绿 `#2DB56B`（二选一作为页面主色，不并列）
- 辅色：浅灰底 `#F0F0F0` 做区域划分
- 强调：暗红 `#C00000` 做核心论点（**用得多但每页不超过1-2句**）
- 装饰：深蓝 `#002060` 做次级权威感（讲者署名、数据来源）

## 三、字体体系

### 字体方案（theme官方）

| 角色 | 字体 |
|---|---|
| 标题字体（西文） | Arial |
| 标题字体（中文） | 微软雅黑 |
| 正文字体（西文） | Arial |
| 正文字体（中文） | 微软雅黑 |

**判断**：所有中文都用 **微软雅黑**，所有西文/数字用 **Arial**（或 Times New Roman 装饰用）。无衬线为主，简洁稳重。

### 字号体系（关键）

| 角色 | 字号 | 用途 |
|---|---|---|
| 封面主标题 | 32pt | 居中、加粗 |
| 页面标题 | 18pt | 左对齐，每页顶部 |
| 章节大字 | 48pt | "CONTENTS"、章节序号 |
| 正文 | 12pt | 主要内容、引导论点 |
| 表格内容 | 12pt 或 10pt | 视密度调整 |
| 图表标题 | 9-10pt | 图下方标注 |
| 数据来源 | 8.25pt | 底部最不显眼位置 |

**关键判断**：字号梯度清晰，**主从层次分明**（32→18→12→9→8.25）。

## 四、版式骨架

### 页面四区结构

```
[0.09-0.61"]  顶部标题区          ← 18pt 微软雅黑左对齐，x=0.29"
─────────────
[1.02-2.20"]  标题下装饰区        ← 浅色块，放红色关键论点（12pt）
─────────────
[2.50-5.00"]  主体内容区          ← 双栏图/表布局常见
─────────────
[5.32-5.62"]  底部数据来源区      ← 8.25pt 标注引用源
```

### 典型版式三种

**A. 双图/双表布局（最高频）**
- 左半：x=0.29", 宽4.4"
- 右半：x=5.30", 宽4.4"
- 中间间距：0.6"
- 用途：对比数据、并列论证

**B. 全宽图 + 全宽文字**
- 图：x=0.34-0.69", 宽8.85-9.15"
- 文字（红色12pt）：在图上方或下方
- 用途：单一核心数据、趋势图

**C. 章节过渡页**
- 左侧深色块：x=-0.02", 宽2.87"，全高（覆盖到边缘）
- 大字"CONTENTS"：48pt 绿色
- "目录"竖排：48pt 嫩绿，分两行
- 右侧空间：放章节标题组（GROUP元素）

### 封面版式

- 主标题（32pt）居中，y=1.74"
- 副标题（讲者+日期，深蓝 `#002060`）居中，y=3.62"
- 极简，无LOGO（这份PPT没角标）

## 五、内容呈现规律

### 论点引导用红色

正文里关键论点用 `#C00000` 红色 12pt 字突出。
例：
> "2024年底我国风电装机规模5.2亿千瓦，新增0.8亿千瓦；光伏装机规模8.9亿千瓦，新增2.8亿千瓦。"

这种关键数据/判断**整段染红**，引导读者抓取核心。

### 数据来源严谨标注

每页底部 `y=5.32"`, `x=0.10"` 位置，8.25pt 小字标注，例：
> "数据来源：中国工程院《电力行业碳达峰碳中和实施路径研究》"

**这是经研院作为智库的专业惯例**——必须保留。

### 图表占比高

28页中，几乎每页都有图、表、图片、AUTO_SHAPE。**纯文字页很少**。

## 六、LOGO使用观察

**这份PPT没有使用LOGO**——封面、内页都没有集团LOGO、经研院LOGO。

**判断**：该参考PPT是**学术分享性质**的讲座，所以省略了LOGO以减少视觉干扰。

**模板设计的处理**：
- 提供 LOGO 占位（左上/右上角），但默认隐藏
- 用户可按场景选择启用：
  - 正式向集团汇报 → 启用 LOGO
  - 学术分享/外部讲座 → 不启用
- 封面始终保留 LOGO 选项

## 七、与今天党课PPT的对比

| 维度 | 党课PPT（朱红金调） | 集团参考PPT（蓝绿调） |
|---|---|---|
| 主色 | 朱红 `#B0151D` | 电投蓝 `#0082D2` |
| 辅色 | 金 `#C9A96E` | 电投绿 `#2DB56B` |
| 字体 | 思源宋体 + 思源黑体 | 微软雅黑 + Arial |
| 装饰 | 顶/底红金渐变条 | 标题下浅色块 |
| 引文 | 红色左竖线 + 浅红背景 | 红色整段染色 |
| LOGO | 左上党徽 + 右上经研院 | 无角标 |
| 用途 | 党建党课（特殊场景） | 集团汇报/研究发布（通用） |

**结论**：党课PPT是**特殊场景**，集团参考PPT是**通用场景**。下一步应该把集团参考的版式作为**标准模板基础**，党课作为派生变体。

## 八、标准模板设计的关键决策

基于以上分析，**标准模板 v1** 的设计将遵循：

### 1. 配色系统（CSS变量）

```css
:root {
  --spic-blue: #0082D2;      /* 主色：电投蓝 */
  --spic-green: #2DB56B;     /* 辅色：电投绿 */
  --spic-blue-light: #2BADFF;
  --spic-green-light: #84C945;
  --emphasis-red: #C00000;   /* 关键论点强调红 */
  --auth-navy: #002060;      /* 深蓝权威感 */
  --neutral-gray: #778495;
  --bg-paper: #FFFFFF;
  --bg-soft: #F0F0F0;
  --ink-black: #000000;
}
```

### 2. 字体策略

```css
font-family: '微软雅黑', 'Microsoft YaHei', 'Noto Sans SC', sans-serif;
/* 数字英文用 Arial */
.num-en { font-family: 'Arial', sans-serif; }
```

### 3. 页面结构（HTML + CSS）

每页一个 `<section class="slide">`，内含四区：

```html
<section class="slide">
  <div class="slide-header">                    <!-- 顶部18pt标题 -->
    <h2>页面标题</h2>
  </div>
  <div class="slide-key-point">                 <!-- 红色关键论点 -->
    <p>... 12pt 红色文字 ...</p>
  </div>
  <div class="slide-content">                   <!-- 主体（图/表/卡片） -->
    ...
  </div>
  <div class="slide-source">                    <!-- 数据来源 -->
    数据来源：xxx
  </div>
</section>
```

### 4. 提供的版式组件

- `.layout-cover` — 封面版式
- `.layout-toc` — 目录/章节过渡版式
- `.layout-dual` — 双栏图/表版式
- `.layout-full` — 全宽图+文字版式
- `.layout-text` — 纯文字论述版式
- `.layout-conclusion` — 结论收口版式

### 5. LOGO 策略

- 提供 LOGO 占位 div，CSS 控制显示/隐藏
- 模板默认：封面显示 LOGO，内页隐藏
- 用户在 HTML 顶部加 `<body class="show-logo">` 即可全局启用
