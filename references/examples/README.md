# Examples · 装配案例

本目录存放使用 SPIC-PPT skill 装配出来的成品案例，供参考和学习。

## product-case/

`ppt-agent-case.html` —— AI 应用中心产品矩阵的 P17 单页，
展示了如何用 v4.3.x 设计语言（.compact 修饰类 + 双栏对齐三件套）做"产品介绍"
类版式。包含：

- key-point 关键论点（顶部）
- 双栏 grid 布局（左：两张特性卡 / 右：装配案例预览）
- preview-wrap（含截图 + 仓库链接）

**这是参考示例，不是 skill 必备资产**——你装配自己的 PPT 时无需引用本目录。
